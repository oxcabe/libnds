Detailed Description

An image abstraction for working with image data.

Image data buffers must be allocated using malloc() rather than pointing to stack data, as the conversion routines will free() the argument's image buffer and allocate a new block for the replacement data. As such, any loader implemented utilizing this structure must use malloc() to allocate the image buffer.
